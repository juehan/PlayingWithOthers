MOTIVATION, What should I get from this class

The trend toward using Web technologies for application development, even outside of the Web browser, is coming along.

As an example, Windows 8 will have provisions for writing apps using the HTML5 technologies. HTML5 will support for things like video streaming, multithreaded and asynchronous processing, direct communications through sockets, and more.

I strongly believe post-pc era has been arrived and mixture of skillset to develop desktop application and web application will be required to most of software developers.

Through this course, I would like to learn web programming as thoroughly as I've wanted to and thus down the line, would like to have working knowledge and hands-on skills in Web development. Ultimately, it will allow me to be well prepared software developer in post-pc era.
